# Changelog

All notable changes to Bonusi (Autoservice) are documented here.

## [1.0.0-rc.1] — 2026-05-29

### ✨ Features — Mobile App
- Шрифт цифр бонусного баланса: **Condensans Paneurope Medium**
- Цифры бонусов позиционированы строго по центру круга (отдельно от подписи)
- SMS OTP аутентификация: вход по номеру телефона через SMSC / SmsAero
- Дефолтный экран входа — SMS (email-вкладка скрыта)
- Плейсхолдеры профиля «ИМЯ» / «МАРКА АВТО» со слайд-ап диалогом заполнения
- Inline просмотр изображений-вложений в чате
- Исправлены относительные URL вложений в мобильном чате

### ✨ Features — Web Admin Panel
- **Ручное начисление/списание бонусов** из карточки клиента
- Текущий бонусный баланс клиента отображается в карточке
- Поиск клиентов по номеру телефона
- Поле VIN автомобиля в карточке клиента
- Кнопка удаления клиента
- Редактирование имени, email, телефона клиента прямо в карточке

### ✨ Features — Backend
- SMS OTP: интеграция SMSC и SmsAero провайдеров
- Endpoint `DELETE /users/:id` для удаления клиента
- Миграция: поле `car_vin` в таблице пользователей
- Database-backed хранилище файлов (таблица `attachment_data`, без S3)
- Endpoint `GET /files/download/:id` для скачивания вложений
- Читаемый номер заказа: формат `ЗН-1001`
- Исправлена транзакционная блокировка в `upsertDevice`

### 🐛 Fixes — Mobile App
- Исправлен краш на старых устройствах (TestFlight build 4)
- Исправлен стиль header в BookingTabScreen (учёт notch)

### 🐛 Fixes — Web Admin Panel
- Авто-пометка сообщений чата прочитанными + очистка счётчика
- Исправлен подсчёт непрочитанных для super_admin
- Исправлен API base URL для вложений в чате
- Исправлены SQL-параметры в `listConversations`

### 🎨 Visual / Style
- Цвет навигационных кнопок и кольца свечения — `#8ECAE6`
- Центровка датчика бонусов по горизонтали по ширине экрана
- Размер шрифта бонусов уменьшен на 50% для корректного отображения
- Удалены декоративные километровые метки и правые дуги с HomeScreen
- Тёмная тема и premium-редизайн страницы входа

### 🏗 Chore / Infrastructure
- Конфигурация SMS OTP переменных в `render.yaml`
- iOS build number: **5**, Android versionCode: **5**
