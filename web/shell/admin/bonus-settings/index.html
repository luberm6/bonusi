Как начислять бонусы

Выберите режим расчёта и сохраните актуальные правила для всех следующих визитов.

—

Бонусы будут начисляться от суммы визита после скидок.

Клиент будет получать фиксированное количество бонусов за каждый визит.

Пример начисления

Если сумма визита 1 000 ₽ → будет начислено 50 бонусов

—

Начисление создаётся автоматически при оформлении визита. Если итоговая сумма визита равна нулю, бонусы не начисляются.

Окончательный расчёт всё равно делает backend, поэтому админский интерфейс показывает только наглядную схему и ожидаемый результат.
